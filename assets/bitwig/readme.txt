ReaBL for Bitwig Beta – Installation

1. Follow the Blender addon installation instructions available on the ReaBL Gumroad page.https://infamedavid.gumroad.com/l/reabl

2. Install the Bitwig extension by dragging and dropping the file
   ReaBlBitwig.bwextension
   into Bitwig.

3. Once installed, add ReaBlBitwig as a controller inside Bitwig.

4. Make sure the IP and port settings match your ReaBL setup in Blender.

After that, ReaBL should be ready to communicate between Blender and Bitwig.
